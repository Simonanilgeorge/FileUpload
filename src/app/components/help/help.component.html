1. Production reports

    Production reports displays production data details with respect to employee information. You can make use of the filters (Date, Select Report) to select the date and type of report you wish to view. You can also make use of filters present inside the table to sort and view data accordingly. Click on the arrow symbol to sort data. eg: Clicking on the sort icon in the employee name column will sort the table in (ascending/descending) order of employee names. Use the export button to export the table data to an excel file.

    1. Daily report

      This page displays production data details over a period of one day.

    2. Monthly report

      This page displays production data details over a period of one month.

    3. Yearly report

      This page displays production data details over a period of one year.

  2. Client reports

    Client reports displays production data details with respect to client information. You can make use of the filters (Date, Select Report) to select the date and type of report you wish to view.

    1. Monthly report

      This page displays production data details over a period of one day.

    2. Yearly report

      This page displays production data details over a period of one year.

    3. DT queue report

      1. Generate DT queue report

        To Generate a DT queue report, upload the RVSI and SP2 files respectively. Once successfully uploaded, the page will automatically redirect to "View DT queue report page".

      2. View DT queue report

        This page by default will show the latest DT queue report generated for the day. You can make use of time and date filters to view previously generated reports.

  3. Admin

    1. Add Employee

      This page is used to add new employees to the application. Production report of a specific employee can only be viewed if he/she is added to the application.

    2. Edit Employee

      This page displays details of all employees. You can filter out employees based on their information by making use of the filters in the table header.

    3. Add Role

      Add Role page will help you add custom roles to the application. Once added you cannot edit the role name.

  4. Order Entry

    Use this page to enter your daily production data. Note: If you have selected Client as NonProd then Order Number, Process, State and Status are not compulsory fields. These fields will default to empty values even if you have entered values for the above mentioned fields.

  5. My Production data

    This page displays list of all production data you have entered. By default it displays production data for the current month (26th of previous month to 25th of current month). You can change the start date and end date to get the production data within the custom time interval. You can either edit or delete the order entries by clicking the respective icons. You can also make use of filters present inside the table to filter out data accordingly. The production data can be exported to an excel sheet by clicking the export button.
