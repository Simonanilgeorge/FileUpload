1. Production reports

    Production reports displays production data details with respect to employee information. Users can make use of two filters (Date,Select Report) to select the date and type of report they want to see eg: Using . Users can also make use of filters present inside the table sort and view data accordingly. Click on the arrow symbol to sort data. eg: Clicking on the sort icon in the employee name column will sort the table in (asceding/descending) order of employee names. Users can click on the export button to export all data to an excel file.

    1. Daily report

      This page displays production data details over a period of one day

    2. Monthly report

      This page displays production data details over a period of one month

    3. Yearly report

      This page displays production data details over a period of one year

  2. Client reports

    Client reports displays production data details with respect to client information. Users have the ability to select two filters (Data,Select Report) to select the date and type of report they want to see.

    1. Monthly report

      This page displays production data details over a period of one day

    2. Yearly report

      This page displays production data details over a period of one year

    3. DT queue report

      1. Generate DT queue report

        To Generate a DT queue report download rvsi and sp2 files from the website and upload it using the buttons present in the card. Once successfully uploaded the page will redirect to "View DT queue report page"

      2. View DT queue report

        This page by default will show the latest generated DT queue report. Users can use the time and date filters to view previously generated reports.

  3. Admin

    1. Add Employee

      This page is used to add new employees. Production data of employees that are not added will not be included in the reports.

    2. Edit Employee

      This page is displays details of all employees added. Users can filter out employees based on their information by making use of the filters in the table header.

  1. Order Entry

    Use this page to enter your daily production data. Note: If you have selected Client as NonProd then Order Number,Process,State and Status are not compulsory fields. These fields will default to empty values even if you have entered values for the above mentioned fields.

  2. My Production data

    This page displays list of all production data you have entered. By default it displays all production data you have entered from 26th of the previous month to 25th of current month. Users can change the start date and end date to get the data entered within that time period. Users can edit or delete order entries by clicking on the respective icons. Users can make use of filters present inside the table to filter out data accordingly. To export the data to an excel sheet click on the export button
